# Examples

This directory contains example codebase guides generated by the **talk-to-codebase** skill.

## Files

### `talk-to-codebase-guide.html`
**Style:** Editorial (default)  
**Generated for:** This repository (talk-to-codebase)  
**Features:**
- Warm dark mode with terracotta and sage green accents
- Magazine-style asymmetric layout
- Inter typography
- Architecture diagram with emoji icons

### `talk-to-codebase-cyberpunk-guide.html`
**Style:** Cyberpunk  
**Generated for:** This repository (talk-to-codebase)  
**Features:**
- Neon terminal aesthetic with cyan/pink/yellow/green colors
- Scanline overlay and CRT flicker effects
- JetBrains Mono monospace font
- Glitch animations on headings
- Terminal-style navigation with $ prefixes
- Architecture diagram with emoji icons

## How to View

Open any HTML file in your browser:

```bash
# macOS
open talk-to-codebase-cyberpunk-guide.html

# Linux
xdg-open talk-to-codebase-cyberpunk-guide.html

# Windows
start talk-to-codebase-cyberpunk-guide.html
```

## Generate Your Own

To generate guides for your own codebase:

```bash
# Navigate to your repository
cd /path/to/your/repo

# Generate with default Editorial style
/talk-to-codebase .

# Or choose a specific style
/talk-to-codebase . cyberpunk style
/talk-to-codebase . bauhaus style
/talk-to-codebase . neumorphism style
```

## Styles Available

- **Editorial** — Warm dark mode, magazine-style (default)
- **Bauhaus** — Industrial geometric design with primary colors
- **Neumorphism** — Soft UI with clay-like tactile interface
- **Cyberpunk** — Neon terminal with CRT effects and glitch animations

See [../references/README.md](../references/README.md) for detailed style comparisons.
